Prompting People

For raw_input, you can also put in a prompt to show to a person so he knows what to type. Put a string that you want for the prompt inside the (). This prompts the user with "Name?" and puts the result into the variable y. This is how you ask someone question and get the answer. This means we can completely rewrite our previous exercise using just raw_input to do all the prompting. 

age = raw_input("How old are you? ")
      height = raw_input("How tall are you? ")
      weight = raw_input("How much do you weigh? ")

      print "So, you're %r old, %r tall and %r heavy." % (
          age, height, weight
      )
